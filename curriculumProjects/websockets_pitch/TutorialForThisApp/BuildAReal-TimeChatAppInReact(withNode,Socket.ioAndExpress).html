How to build a real time chat app in React (with node, socket.io and express)

jelo rivera
Oct 17, 2018 · 4 min read

Have you ever wondered how some websites give you live updates so fast?

Facebook’s Messenger sends messages as soon as they are sent and you even receive those messages on slow networks. The answer to the question is simple. WEB SOCKETS.

This article will teach you how to use web sockets in a real life app. This article won’t teach you about how web sockets work, but if that is something you’re interested in, you can read more about it here.

In our demo, a server will handle connections and assign every connection a username. The connections share one “pot”. The pot increases in value if someone pitches in and decreases if someone clicks the “get one” button. Everybody who is connected gets notified if someone pitched in or someone got one! The notification is real time and the pot’s value also gets updated. You can find the finished source code here.

In Conclusion 💭

Web sockets is a very powerful tool that enables fast and reliable real time data exchange for both the client side and server side of the web. This article was created to exhibit just that.
